## 프로젝트 개요

### 기획 배경

- 2D 교재의 한계: 평면 도면으로 공간감 파악이 어려움
- 실습 부재: 실제 부품 분해·조립 학습 환경 부족
- AI 활용: 맞춤형 질의응답 기반 학습 필요

**타겟 사용자**: 기계공학 전공 학생, 공학도, 기술 교육 기관

## 기술 스택

Three.js · React Three Fiber · Zustand · React Query · Tailwind · Radix UI

## 주요 기능

- **3D 모델 뷰어** — GLB 자동 로딩, OrbitControls 회전, 분해 애니메이션, 부품 선택 & 하이라이트
- **AI 학습 어시스턴트** — 부품 상세 설명, 실시간 질의응답, 스트리밍 응답, 학습 맥락 유지
- **퀴즈 & 학습 도구** — 객관식/주관식, 개인 메모, PDF 다운로드, 학습 진행률 추적
- **부품 탐색 시스템** — 계층 구조 목록, 마크다운 렌더링, 무한 스크롤, 상세 검색 필터

## 트러블슈팅

### 1. 3D 모델 분해(Explode) 애니메이션

각 부품을 자연스럽게 분리하고 슬라이더로 실시간 조절해야 했습니다. 좌표 하드코딩은 모델마다 재작업이 필요했고, 단순 중심점 계산은 부자연스러웠습니다.

- 각 부품에 방향 벡터 & 거리 **메타데이터** 설정
- 슬라이더(0~1)에 따라 선형 보간 계산
- Three.js `Vector3`로 부드러운 애니메이션 → 5개 모델 모두 자연스러운 분해/조립 완성

### 2. Three.js Canvas 내외부 상태 동기화

부품 선택·분해·카메라 상태 등 복잡한 상태를 Canvas 내부/외부 React UI 간 공유하기 어려웠고, 리렌더링 시 렌더링 성능이 저하됐습니다.

- Zustand로 뷰어 상태를 React **외부에서 중앙 관리**
- Selector로 필요한 상태만 구독 → 리렌더링 최적화
- Persist 미들웨어로 사용자 데이터 자동 저장
